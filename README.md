#  Propvision: AI Property Issue Diagnosis System

An intelligent, multimodal AI platform designed to automate property defect classification and streamline maintenance workflows. By analyzing both image and text data, the system diagnoses structural, cosmetic, or maintenance issues, significantly reducing the need for manual inspections.

---

##  Key Features

* **Multimodal Data Analysis:** Jointly processes image uploads and textual descriptions to accurately classify property defects with an accuracy rate of 85%+.
* **Smart Triaging:** Automatically engineers severity and confidence metrics for every detected issue, reducing manual verification pipelines by 30%.
* **Human-in-the-Loop (HITL):** Routes low-confidence or high-severity predictions to a dedicated validation queue for expert review, ensuring high reliability.
* **Modern Architecture:** Built with a scalable decoupled structure featuring optimized backend systems and a highly responsive user interface.

---

##  Tech Stack & Skills Used

* **Frontend:** React, Node.js, Modern UI component libraries
* **Backend:** TypeScript / Node.js
* **AI/ML Core:** Python, Multimodal Transformers, Computer Vision (CNNs), Natural Language Processing (NLP)
* **Data Processing:** Feature Engineering, Automated Severity Scoring

##  Getting Started

### 1. Repository Setup

Clone the repository and jump into the project directory:

```bash
git clone [https://github.com/KARTHIGADATASCIENCE/Propvision---AI-.git](https://github.com/KARTHIGADATASCIENCE/Propvision---AI-.git)
cd "Propvision---AI-"

```

### 2. Backend Setup (TypeScript)

Navigate to the backend directory, install the server dependencies, and start the development server:

```bash
cd backend-ts
npm install
npm run dev

```

### 3. Frontend Setup

Open a new terminal window, navigate to the frontend folder, install the UI dependencies, and launch the application:

```bash
cd frontend
npm install
npm start

```
